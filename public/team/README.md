# Team headshots

Drop founder photos here, named by the member `slug` from
`lib/presentation/content.ts` (the `team.members[].slug` field):

- `alexander.jpg`
- `vladislav.jpg`
- `dimitar.jpg`
- `mario.jpg`

Square images work best (they're cropped to a circle, rendered at 56×56). If a
file is missing, the deck automatically falls back to an initials avatar — so the
slide always looks intentional.
